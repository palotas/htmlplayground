Selenium Introduction Training

Date: on request

Location: worldwide

Language: English or German

INTRODUCTION

Selenium is an open source browser automation framework and quickly becoming the world’s number one solution for test automation. The tool allows for highly scalable test automation solutions at with a very low maintenance effort. Selenium has the support of some of the largest browser vendors who have taken (or are taking) steps to make Selenium a native part of their browser. It is also the core technology in countless other browser automation tools, APIs and frameworks.

GOALS

Gain a good understanding of the Selenium automation toolset. Learn how and when to use the different flavors of Selenium (IDE, WebDriver, GRID). All topics are accompanied by practical examples and real life experiences.

CONTENT

  • Introduction into Selenium and the world of open source
  • A word on regression testing
  • What is Selenium and what is Selenium for?
  • Write tests with Selenium IDE and Selenium Webdriver
  • Selenium GRID - run tests in parallel and dramatically reduce execution times
  • The maintenance aspect - how to bring your maintenance to a minimum
  • Scaling your tests

TARGET AUDIENCE

  • Testers who want to gain an insight into test automation with Selenium or who are currently working on automation and want to learn more about Selenium and the use of open source toolsets
  • Test Leads and Managers want to get a solid introduction to Selenium and who want to learn about using using open source toolsets in their organizations
