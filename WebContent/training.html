INTRODUCTION

Selenium is an open source browser automation framework and quickly becoming the world’s number one solution for web test automation. The tool allows for highly scalable test automation solutions with a very low maintenance effort and covers all browsers. Some of the largest companies on the web are using and actively supporting Selenium. Selenium has also the support of some of the largest browser vendors who have taken (or are taking) steps to make it a native part of their browser. It is also the core technology in countless other browser automation tools, APIs and frameworks.

GOALS

Gain a good understanding of the Selenium automation toolset. Learn how and when to use the different flavors of Selenium (IDE, WebDriver, GRID). Learn how to write efficient tests that require low maintenance effort. All topics are accompanied by practical examples and real life experiences.

CONTENT

  • Introduction to Selenium and test automation with open source tool sets
  • Automated regression testing - what should be automated and what should not
  • The Selenium tool family - understand the different flavors of Selenium
  • Write tests with Selenium IDE and Selenium WebDriver
  • Selenium GRID - design and write tests for parallel execution in Selenium Grid
  • The maintenance aspect - how to bring your maintenance down to a minimum
  • Scaling your tests

TARGET AUDIENCE

  • Development oriented testers, developers, test leads and test managers who want to get a solid introduction to Selenium and gain a practical insight into test automation with the number one open source automation tool

PREREQUISITES

  • Basic Java skills
