SERVICES AND SOLUTIONS

GRIDFUSION is specialized in test automation with open source toolsets mainly around Selenium and Webdriver. We cover the entire value chain from the first assessment of your organisation regarding the use of open source automation tools, accompanied by training on how to use them properly all the way to the actual implementation of the solution together with your team.

Test Automation Assessments

Test automation assessments are a great way to get a good insight into your current state of test automation. These engagement typically go from 2-5 days onsite at your company. The result is as in depth assessment on the current state and areas for improvement with step by step instructions on how to get there.
Areas of focus include:
  • Test environment setup and configuration
  • General test automation approach
  • Proper use of modularization (i.e. page objects)
  • Code reviews
  • Virtualization
  • Selenium GRID configuration
  • Analysis of instable tests
