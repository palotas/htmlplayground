Selenium ist eines der populärsten Open Source Browser Automatisierungs Framework, das hoch skalierbare Testautomatisierungslosüngen mit einem sehr geringen Wartungsaufwand ermöglicht. Einige der grössten Unternehmen im Web verwenden und unterstützen Selenium. Ebenso hat Selenium auch die Unterstützung der grössten Browser Hersteller, die Schritte unternommen haben, um Selenium einen nativen Teil ihres Browsers zu machen.

ZIELE
  • Lernen Sie, wann und wo sich der Einsatz der Selenium Tools eignet (IDE, Selenium 2 /Webdriver, Selenium GRID)
  • Lernen Sie effiziente, skalierbare Tests mit einem niedrigen Wartungs Aufwand zu planen und zu schreiben
INHALT
  • Einführung in Selenium und Open Source Automatisierungs Tools
  • Automatisierung - was sollte und was sollte nicht automatisiert werden
  • Die Selenium Tool Familie, lernen Sie die unterschiedlichen Tools kennen
  • Page Objects
  • Selenium GRID - parallele Testausführung
  • Praktische Übungen und Coding Sessions
  • Wartung - wie kann man die Wartung auf ein minimum reduzieren?
  • Skalierung der Tests
  • Alle Themen werden durch praktische Übungen und Beispiele begleitet
ZIELGRUPPE
  • Test Engineers, Test Leads- und Manager, Entwickler
VORAUSSETZUNGEN
  • Grundlagen in Programmierung (idealerweise JAVA)
INKLUSIVE
  • Dokumentation (Hardcopy + elektronisch), Source Code der Übungen, Verpflegung (Kaffee, Mittagessen, Snack)
